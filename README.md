# BMC WebUI 开发指导

本网站结合 **OpenBMC WebUI** 与 **AMI BMC WebUI** 的技术栈，介绍了前端 **基础（HTML/CSS/JS）、框架（Vue/AngularJS）、工程化、浏览器调试技巧、图标、UI 组件库、图表库（echarts 等）、UI 图设计** 等知识，帮助开发人员快速上手 BMC WebUI 开发。

| 周五 7 月 11 日 | 前端开发基础 ②                | - JS 基础语法<br>- 浏览器存储<br>- 简单交互逻辑演示                                                 |
