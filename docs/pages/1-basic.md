## 页面运行机制

> 目标：理解“前端是怎么跑起来的”。

* 浏览器的工作流程：

  * 输入网址 → 浏览器发送 HTTP 请求

  * 接收 HTML → 渲染页面结构

  * 下载 CSS → 渲染页面样式

  * 下载 JS → 执行页面逻辑

* DOM（页面结构） + CSSOM（样式结构）组成最终页面

* 简单流程图辅助理解（不用讲细节）

* 前端和后端的分工（前端负责“显示”和“交互”，后端负责“数据”）
